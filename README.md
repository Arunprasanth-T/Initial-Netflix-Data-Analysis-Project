🎬 Netflix Data Analysis using Python
📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python.

The objective is to clean the dataset, analyze trends, answer business questions, and generate meaningful insights using data visualization.

🎯 Project Objectives
Understand the Netflix content library
Clean and preprocess real-world data
Analyze trends in movies and TV shows
Identify popular genres, countries, directors, and actors
Generate business insights from the dataset
📂 Dataset

Dataset Name

Netflix Movies and TV Shows Dataset

Source

Kaggle

🛠️ Technologies Used
Python
Pandas

Matplotlib
Jupyter Notebook
📋 Data Cleaning

The following preprocessing steps were performed:

Removed duplicate records
Handled missing values
Converted date_added into datetime format
Created a new column (year_added)
Cleaned text columns
Processed multi-valued columns using split() and explode()
📊 Business Questions
1. Movies vs TV Shows
Compared the number of Movies and TV Shows available on Netflix.
2. Top Producing Countries
Identified countries contributing the most Netflix content.
3. Content Added Over the Years
Analyzed Netflix's content growth over time.
4. Most Popular Genres
Determined the most common genres available.
5. Distribution of Content Ratings
Examined the distribution of maturity ratings.
6. Movie Duration Analysis
Studied the distribution of movie durations.
7. Top Directors
Found directors with the highest number of titles.
8. Top Actors
Identified actors appearing most frequently.
9. Monthly Content Addition
Analyzed which months Netflix adds the most content.
10. Release Year Analysis
Examined trends in content release years.
📈 Key Insights
Movies dominate Netflix's content library.
The United States and India contribute the largest number of titles.
Netflix experienced rapid content growth between 2017 and 2020.
International Movies and Dramas are the most common genres.
Most movies have a duration between 90–120 minutes.
Popular directors include Youssef Chahine, Martin Scorsese, and Steven Spielberg.
Actors such as Anupam Kher and Shah Rukh Khan appear frequently.
Netflix primarily hosts modern content released after 2000.
